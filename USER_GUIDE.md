# USER_GUIDE.md

## 概要
このアプリは、iPhone にある写真や画像を選び、A4 印刷向けの PDF にまとめるためのものです。
画像を選んで、並べ方を決めて、プレビューを見てから PDF を作れます。

## 使う前に
- iPhone で `index.html` を開いて使います。
- 画像は `JPEG` と `PNG` を選べます。
- 共有できない場合に備えて、保存先を選べる状態にしておくと安心です。
- 画像枚数が多いと、読み込みや PDF 生成に時間がかかることがあります。

## 使い方
1. `Screen A` で画像を選びます。
2. `Screen B` で分割レイアウトと表示方法を選びます。
3. `Screen C` で仕上がりを確認し、`PDFを生成` を押します。
4. `Screen D` で `共有する` または `保存する` を選びます。

## 画面ごとの説明
### Screen A: 画像を選ぶ
- `画像を選択` を押して、印刷したい画像を選びます。
- 選んだ画像は一覧で表示されます。
- 選択後に次へ進みます。

### Screen B: 並べ方を決める
- A4 に何枚並べるかを `1 / 2 / 4 / 6 / 8` 分割から選びます。
- `全体表示` は画像全体を見せたいときに向いています。
- `トリミング` は余白を減らしたいときに向いています。

### Screen C: 仕上がりを確認する
- A4 に並んだ状態を確認できます。
- 画像が複数ページになるときは、ページ送りで内容を見られます。
- 問題がなければ `PDFを生成` を押します。

### Screen D: 保存または共有する
- 作成した PDF の名前とページ数を確認できます。
- `共有する` が使えるときは、共有シートを開けます。
- `共有する` が使えないときは、`保存する` を使って PDF を保存してください。
- `PDFを開いて確認` で PDF の見え方を確認できます。

## 困ったとき
### 画像を選んだのに進めない
- 画像の読み込みに少し時間がかかることがあります。
- しばらく待っても変わらない場合は、画像を選び直してください。

### PDF生成に時間がかかる
- 画像が多い、または高解像度だと時間がかかります。
- 使う画像を減らして再度試してください。

### 共有できない
- 端末やブラウザによっては共有機能が使えないことがあります。
- その場合は `保存する` を選んでください。

### 思ったより切り抜かれる
- `トリミング` を選ぶと、画像の端が見切れることがあります。
- 全体を残したい場合は `全体表示` に戻してください。

## 既知の制約
- 画像が多すぎると、ブラウザが途中で重くなることがあります。
- ブラウザによって PDF の開き方や見え方が少し違うことがあります。
- 並び順の細かい入れ替えや余白の手動調整には対応していません。
